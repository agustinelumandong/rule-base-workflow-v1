# Western Dialogue Rules

## Goal

Western dialogue should sound plainspoken, tense, and character-specific without becoming cartoon speech.

## Required Behavior

- Keep lines concise where pressure is high.
- Let silence, hesitation, object handling, and refusal carry meaning.
- Use period-aware vocabulary only when approved for the project.
- Match each line to the character profile, social position, education, region, and emotional restraint.
- Let dialogue reveal character, build tension, advance plot, or expose relationship pressure.

## Dialect Policy

Allowed:

- Light contractions.
- Limited dropped auxiliaries.
- Regional or occupational vocabulary where approved.
- Rougher rhythm for characters whose profile supports it.

Blocked:

- Heavy phonetic spelling.
- Repetitive apostrophes.
- Generic parody words.
- Modern slang.
- Over-polished speech for characters whose life and context do not support it.
- Dialect used to mock or flatten a culture, class, or ethnicity.

## Examples

Weak:

```text
He was very angry and wanted revenge.
```

Better:

```text
He set the cup down without a sound. Then he reached for his hat.
```

Too polished:

```text
What are you talking about?
```

More direct, if supported by character voice:

```text
What you talking about?
```

Do not force this pattern onto every character. Grammar is part of character governance, not a decoration.
