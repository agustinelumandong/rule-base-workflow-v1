# Western Prompt Blocks

These blocks are reusable fragments for BookForge prompts. They are not workflow authority; they operate under CanonCore, PolicyCenter, MemoryVault, AuthenticityPolicy, and ContinuityLockPolicy.

## Prime Directive Block

```text
Write this Western scene under the Western Manuscript Prime Directive:
the manuscript must remain consistent, physical, era-accurate, and character-driven.

Show meaning through action, silence, movement, tools, horses, weapons, weather,
distance, work, scarcity, and consequence. Do not explain what the scene has already
shown. Do not introduce unsupported canon, modern language, or wrong-period details.
```

## Setting Block

```text
Make the setting active. The land, weather, distance, animals, tools, supplies,
and social pressure must influence what characters do. Do not use the frontier
as wallpaper.
```

## Dialogue Block

```text
Make dialogue concise, period-aware, and character-specific. Avoid modern phrasing,
over-polished grammar, fake cowboy parody, heavy phonetic spelling, and excessive slang.
Use dialect lightly and only when supported by the character profile.
```

## Theme Block

```text
Express Western theme through action and cost. Justice, survival, loyalty, betrayal,
redemption, freedom, law, and violence must appear through choices and consequences,
not speeches or abstract moral summary.
```

## Character Conflict Block

```text
Drive the scene from character goals, wounds, fears, loyalties, moral code, and pressure.
The protagonist and antagonist should both want something concrete. Supporting characters
must have a purpose beyond decoration.
```

## Revision Block

```text
Revise for Western force without changing canon. Strengthen physical grounding,
directness, dialogue restraint, setting pressure, and consequence. Preserve all approved
facts, character state, relationships, timeline, object state, and numeric locks.
```

## Validator Instruction Block

```text
Evaluate whether the passage remains consistent, physical, era-accurate, and
character-driven. Report findings with evidence, severity, violated rule, and recommended
action. Do not hide lower-level issue IDs behind the Western prime directive rollup.
```
