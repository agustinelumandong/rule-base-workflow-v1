# Western Prompt Blocks

Use these reusable instruction blocks in prompt generation templates to enforce genre parameters in writer and editor personas.

## Setting Block (`western_setting_prompt`)
```markdown
Make the setting active. The land, weather, distance, animals, tools, and scarcity should influence what characters do. Do not use the setting as wallpaper. Focus on sensory grounding: alkaline dust, dry wind, scent of pine needles, creaking leather.
```

## Dialogue Block (`western_dialogue_prompt`)
```markdown
Rewrite dialogue to sound period-aware, concise, and character-specific. Avoid modern speech, over-polished phrasing, and fake cowboy parody. Use idioms sparingly. Anchor speech with physical action anchors using em dashes rather than standard speech tags.
```

## Theme Block (`western_theme_prompt`)
```markdown
Strengthen Western themes through action: justice, redemption, survival, loyalty, betrayal, freedom versus civilization, law versus lawlessness. Do not explain the theme directly; let moral conflicts emerge entirely from character decisions and their consequences.
```
