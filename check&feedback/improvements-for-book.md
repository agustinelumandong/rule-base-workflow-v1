# Western Prose Style Guide

Compiled from the screenshots. Duplicate **Section 8: Pacing Through Action** has been included once.

---

## 1. Character Voice Differentiation

| Character          | Voice Rule                                                 | Example                                          |
| ------------------ | ---------------------------------------------------------- | ------------------------------------------------ |
| **Hunter Jones**   | Complete statements, no questions, laconic, controlled     | “It is enough.” not “Is it enough?”              |
| **Lila Whitaker**  | Shorter, more direct, commands, practical                  | “Then saddle him right.”                         |
| **Elias Whitaker** | Slightly informal grammar, measured, grounded              | “That don’t make it light.”                      |
| **Len Wood**       | Controlled threats, no long villain speeches, repeats back | Repeats part of what others say before answering |
| **Bounty Hunter**  | Restrained, factual, no exposition dumps                   | States facts without emotional coloring          |

**Principle:** Each character speaks differently. Dialogue reveals personality without author explanation.

---

## 2. Tone & Voice Lock

| Rule                                 | What It Means                                        |
| ------------------------------------ | ---------------------------------------------------- |
| **Literal prose only**               | No metaphors, similes, or personification            |
| **Behavior over thought**            | Show what characters **do**, not what they **think** |
| **No internal monologue**            | No “he thought,” “he realized,” “he wondered”        |
| **Short sentences during action**    | Choppy rhythm = tension                              |
| **Mixed sentence lengths elsewhere** | Uneven rhythm = natural voice                        |
| **No modern vocabulary**             | Period-authentic 1878 language only                  |

**Principle:** The narrator never interprets for the reader. Actions speak. Words carry weight.

---

## 3. Action Over Saying It

| Bad / Telling         | Good / Showing                                                                          |
| --------------------- | --------------------------------------------------------------------------------------- |
| “He was nervous”      | “His hand stayed above the revolver, fingers bent and ready.”                           |
| “She was angry”       | “Her jaw tightened. She stepped down to the yard.”                                      |
| “He felt pain”        | “The shoulder screamed. He ignored it.”                                                 |
| “The town was afraid” | “Two women near the cook fires gathered their iron pots and went under canvas.”         |
| “She cared about him” | “She walked beside him, close enough that their boots hit the dirt in the same rhythm.” |

**Principle:** Never state an emotion. Show the physical behavior that reveals it.

---

## 4. Emotional Restraint

| Rule                            | What It Prevents                              |
| ------------------------------- | --------------------------------------------- |
| **No “he felt” + emotion**      | Avoids stating feelings directly              |
| **No “he realized” + truth**    | Avoids author intrusion into character’s mind |
| **No explanation after action** | Behavior already shows what’s happening       |
| **No pleading speeches**        | Characters act, not explain                   |
| **No stated love/affection**    | Shared work and presence = relationship       |

**Example:** Lila doesn’t say, “I care about you.” She changes his bandages every morning without talk. She walks beside him. She says, “Ride straight.” That’s the emotion.

---

## 5. Sensory Grounding

| Sense     | What We Used                                           |
| --------- | ------------------------------------------------------ |
| **Sight** | Physical details only: dust, sweat, blood, metal, wood |
| **Sound** | Hooves on stone, leather creaking, rifle report, wind  |
| **Smell** | Manure stink, coal smoke, cedar smoke, pine rot        |
| **Touch** | Cold metal, rough wood, hot barrel, wet horse neck     |
| **Taste** | Coffee grounds, salt pork, bitter beans                |

**Principle:** Sensory details = immersion. Never describe what a character sees without also hearing, smelling, or feeling something.

---

## 6. Western Prose Style

| Rule                    | What It Means                                               |
| ----------------------- | ----------------------------------------------------------- |
| **Plain vocabulary**    | Iron, leather, dirt, lead, bone, granite                    |
| **No Texas slang**      | Authentic 1878 range talk, not movie dialogue               |
| **Mechanical accuracy** | Winchester lever action, Colt cylinder check, reloading     |
| **Physical verbs**      | “Hit” not “opened” for pain; “drove” not “pushed” for force |
| **No poetic lines**     | Plain statements, no figurative language                    |

**Example:** Instead of “The pain was like fire,” write: “The pain was a hot wire running from his neck to his elbow.”

---

## 7. Scene Economy

| Rule                                | What It Prevents                  |
| ----------------------------------- | --------------------------------- |
| **First-line indent on paragraphs** | Visual paragraph separation       |
| **Scene breaks as * * ***           | Clean transition markers          |
| **Page breaks between chapters**    | Professional manuscript structure |
| **No filler paragraphs**            | Every paragraph moves the story   |
| **No repeated information**         | Trust the reader to remember      |

**Principle:** Cut anything that doesn’t advance plot, reveal character, or build tension.

---

## 8. Pacing Through Action

| Technique                         | How It Works                                              |
| --------------------------------- | --------------------------------------------------------- |
| **Short sentences = tension**     | “Hunter fired once. The shot cracked against the stone.”  |
| **Long sentences = observation**  | Detailed tracking, landscape, horse care                  |
| **Sentence fragments for impact** | “Five now. Len Wood. The second-in-command.”              |
| **Repetition for rhythm**         | “Not gone. Nothing was ever gone to a man who knew sign.” |
| **White space for breathing**     | Single-line paragraphs = emphasis                         |

**Principle:** Rhythm = pacing. Short = fast. Long = slow. Fragments = punch.

---

## 9. Dialogue as Action

| Rule                                     | What It Means                                        |
| ---------------------------------------- | ---------------------------------------------------- |
| **No dialogue tags when action anchors** | “Walk.” He turned. Not: “Walk,” he said angrily.     |
| **Dialogue reveals, doesn’t explain**    | Characters say what they’d actually say              |
| **Short exchanges = tension**            | Quick back-and-forth = conflict                      |
| **Silence = character**                  | What characters **don’t** say matters                |
| **No exposition through dialogue**       | Characters don’t explain what they both already know |

**Principle:** Dialogue is action. Each line should do something, not just convey information.

---

## 10. Continuity as Craft

| Rule                       | What It Prevents                                                |
| -------------------------- | --------------------------------------------------------------- |
| **Wound tracking**         | Shoulder graze → thigh graze → side stab → healed by Chapter 10 |
| **Gang count progression** | 6 → 5 → 3 → 3 → 2 → 2 → 0                                       |
| **Time awareness**         | 14 days main action, 3 months gap, first snow                   |
| **Physical consistency**   | Hunter’s injuries affect his movement in every chapter          |
| **Weapon consistency**     | Winchester always lever-action, Colt always cylinder-checked    |

**Principle:** The reader trusts the story when details stay consistent.

---

## 11. Emotional Impact Without Stating It

| Emotion        | How It’s Shown — Never Stated                                    |
| -------------- | ---------------------------------------------------------------- |
| **Trust**      | Lila changes bandages without talk; Hunter shares tactical plans |
| **Respect**    | Elias brings food, stands in doorway, leaves when Hunter eats    |
| **Fear**       | Women gather pots and go under canvas; boy stops running         |
| **Duty**       | Hunter takes the badge, looks at the road, stays                 |
| **Connection** | Boots hit dirt in same rhythm; hands close but not touching      |
| **Loss**       | Lila doesn’t wave; Hunter doesn’t raise his hand                 |

---

## 12. Hidden Weakness Principles

| Principle                    | What It Addresses                                |
| ---------------------------- | ------------------------------------------------ |
| **Physical interiority**     | Breath, body response, sensation — never thought |
| **Dialogue distinctiveness** | Each character has a unique speech pattern       |
| **Repetition reduction**     | Vary openers, avoid pattern tics                 |
| **Sensory depth**            | Add taste, smell, texture at key moments         |
| **Landscape pressure**       | Setting forces decisions, not just decorates     |
| **Quiet humanizing beats**   | Shared work before crisis = emotional weight     |

---

# Summary: The 5 Core Rules

1. **Show, don’t tell** — Behavior > emotion. Action > explanation.
2. **Voice matters** — Each character speaks differently.
3. **Plain prose** — No metaphors, no similes, no personification.
4. **Physical grounding** — Every scene has sound, smell, touch, or taste.
5. **Earned emotion** — Relationships are built through shared work, not stated feelings.

---

These principles turned a 6.5/10 manuscript into a 9.0/10 within phase-0 constraints. The book reads like a Western because it follows Western prose rules, not literary fiction rules.
